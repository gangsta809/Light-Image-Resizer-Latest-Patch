# Light Image Resizer

## Overview

Light Image Resizer is a Windows-based application designed to simplify the process of resizing, converting, and organizing images. It aims to assist users in managing large collections of photos by providing a straightforward interface for adjusting image dimensions and formats efficiently.

## Key Features

**Resize images in multiple formats**  
**Batch processing for handling numerous files simultaneously**  
**Convert images between popular file types**  
**Adjust image quality and resolution with precision**  
**Rename files during processing for better organization**  
**Create copies of images without altering originals**  
**Support for common image formats including JPEG, PNG, BMP, and TIFF**  
**Simple drag-and-drop functionality for ease of use**
